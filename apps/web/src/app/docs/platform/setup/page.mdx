# Setup

Get started with creating your Karrio Platform account and setting up your integration.

## 1. Sign up for Karrio Platform

Head over to the [Karrio Platform](/platform) and create a new account by clicking on the **Get Started** button.

![Sign up for Karrio Platform](/placeholder.svg)

## 2. Create your Platform Team

After signing up, you'll be prompted to create your platform team. This is where you'll manage your integration settings and API keys.

![Create Platform Team](/placeholder.svg)

## 3. Configure API Keys

Once your platform team is created, you'll need to generate your API keys:

1. Go to your platform settings
2. Navigate to the API Keys section
3. Generate a new API key
4. Store your API key securely

![Configure API Keys](/placeholder.svg)

## 4. Set up Carrier Connections

Connect your preferred carriers to your Karrio Platform account:

1. Go to the Carrier Connections section
2. Select the carriers you want to integrate
3. Follow the carrier-specific setup instructions
4. Test your connections

![Set up Carrier Connections](/placeholder.svg)

## 5. Configure Webhooks (Optional)

If you want to receive real-time updates about your shipments, set up webhooks:

1. Go to the Webhooks section
2. Add your webhook endpoint
3. Select the events you want to receive
4. Test your webhook configuration

![Configure Webhooks](/placeholder.svg)

## 6. You're Ready!

Once you've completed these steps, you're ready to start using Karrio Platform. Head over to our [Quickstart Guide](/docs/platform/quickstart) to learn how to make your first API call.
