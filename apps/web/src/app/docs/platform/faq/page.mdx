# FAQ

## General Questions

### What is Karrio Platform?

Karrio Platform is a comprehensive shipping API that enables businesses to integrate multi-carrier shipping capabilities into their applications. It provides a unified interface for connecting with multiple carriers, generating shipping labels, tracking shipments, and managing shipping workflows.

### Who is Karrio Platform for?

Karrio Platform is designed for:
- Logistics providers
- E-commerce platforms
- Retailers
- 3PLs (Third-Party Logistics)
- Marketplaces
- Any business that needs shipping functionality

### What carriers does Karrio Platform support?

Karrio Platform supports a wide range of carriers including but not limited to:
- UPS
- FedEx
- DHL
- USPS
- Canada Post
- And many more

## Technical Questions

### How do I get started with Karrio Platform?

To get started:
1. Sign up for a Karrio Platform account
2. Create your platform team
3. Generate your API keys
4. Connect your preferred carriers
5. Start making API calls or using our SDK

### What programming languages are supported?

Karrio Platform provides:
- A RESTful API that can be used with any programming language
- Official SDKs for JavaScript/TypeScript
- React components and hooks for web applications

### How do I handle errors in the API?

The API returns standard HTTP status codes and detailed error messages. All errors include:
- An error code
- A descriptive message
- Additional context when available

### Is there a rate limit?

Yes, Karrio Platform has rate limits based on your subscription plan. The limits are:
- Free tier: 100 requests per minute
- Pro tier: 1000 requests per minute
- Enterprise tier: Custom limits

## Integration Questions

### How do I integrate Karrio Platform into my application?

You can integrate Karrio Platform in several ways:
1. Using our REST API directly
2. Using our JavaScript/TypeScript SDK
3. Using our React components and hooks
4. Using our pre-built UI components (Elements)

### How do I handle webhooks?

Karrio Platform supports webhooks for real-time updates. To set up webhooks:
1. Configure your webhook endpoint in the platform settings
2. Select the events you want to receive
3. Implement the webhook handler in your application
4. Test your webhook configuration

### How do I handle international shipping?

Karrio Platform provides comprehensive support for international shipping:
- Customs documentation generation
- HS code validation
- Duties and taxes calculation
- International address validation

## Billing Questions

### How is pricing determined?

Pricing is based on:
- Number of API requests
- Number of shipments
- Number of carrier connections
- Additional features used

### What payment methods are accepted?

We accept:
- Credit cards (Visa, Mastercard, American Express)
- Bank transfers
- PayPal

### Can I change my subscription plan?

Yes, you can upgrade or downgrade your subscription plan at any time. Changes will take effect at the start of your next billing cycle.
